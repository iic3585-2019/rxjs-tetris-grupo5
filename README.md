## Instrucciones de ejecución

Ejecute el comando

```
npm install
```

Luego de que se instalen todas las dependencias del proyecto, ejecute el siguiente comando

```
npm run start
```

Finalmente para jugar dirigase a la url `http://localhost:8080/`
